# Hello World - 我的个人网站上线了

欢迎来到我的个人空间！

## 关于这个网站

这是一个使用 Astro + Fuwari 模板搭建的个人网站，部署在 Cloudflare Pages。

## 技术栈

- **框架:** Astro 5.13.10
- **样式:** Tailwind CSS 3.4.19
- **部署:** Cloudflare Pages
- **搜索:** Pagefind
